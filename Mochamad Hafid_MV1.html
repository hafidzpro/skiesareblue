Analisis Hasil Pengisian Missing Values

1. Pengisian Missing Values pada Kolom Objek

  • df.select_dtypes(include='object').mode().iloc[0] mengambil nilai mode (nilai yang paling sering muncul) dari setiap kolom bertipe objek (string/kategori).
  • df.fillna(..., inplace=True) menggantikan missing values dalam kolom tersebut dengan nilai mode.
  • Hasil: Semua missing values dalam kolom kategori diisi dengan nilai yang paling sering muncul di kolom masing-masing.

2. Pengisian Missing Values pada Kolom Numerik

  • df.select_dtypes(include='number').mean().iloc[0] mengambil mean (rata-rata) dari setiap kolom bertipe numerik.
  • df.fillna(..., inplace=True) menggantikan missing values dalam kolom numerik dengan nilai mean.
  • Hasil: Semua missing values dalam kolom numerik diisi dengan rata-rata masing-masing kolom.
